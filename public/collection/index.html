B.Sc. (Hons.) R modules

Landing page for BCB’s R workshops.

These workshops are scheduled to take place during the first semester each year. BCB744 is a core module and will be taken by all B.Sc. (Hons.) students. BCB743 is an elective module – please discuss with your supervisor if taking this module will benefit you.
